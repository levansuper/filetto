# Filetto File Server

## How To Install:
`sudo apt-get install imagemagick` <br>
`npm install filetto`<br>
`npm install`<br>

###change:
`config/servers/web.js` to change port<br>
`config/mongoose.js` to change mongodb connection<br>


## To Run:
`npm start`


## API documentation
http://localhost:3050/api/showDocumentation
